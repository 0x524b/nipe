#### Download and install:
```
    git clone https://github.com/GouveaHeitor/nipe
    cd nipe
    cpan install Switch JSON LWP::UserAgent
```

#### Commands:
```
    COMMAND          FUNCTION
    install          install dependencies
    start            start routing
    stop             stop routing
    status           see status

    Examples:

    perl nipe.pl install
    perl nipe.pl start
    perl nipe.pl stop
    perl nipe.pl status
```
