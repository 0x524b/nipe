***Script to redirect all the traffic from the machine to the Tor network.***

####Installing:

    git clone https://github.com/HeitorG/nipe
    cd nipe
    cpan install  strict warnings Switch


####Commands:

	COMMAND          FUNCTION
	install          To install.
	start            To start
	stop             To stop
