####Installing:
```
    git clone git@github.com:HeitorG/nipe.git
    cpan install  Switch Mojo::UserAgent
```

####Commands:
```
    COMMAND          FUNCTION
    install          to install
    start            to start
    stop             to stop

    Examples:

    perl nipe.pl install
    perl nipe.pl start
    perl nipe.pl stop
```
