####Installing:
```
    git clone https://github.com/HeitorG/nipe
    cd nipe
    cpan install  Switch WWW::Mechanize LWP::Protocol::https
```

####Commands:
```
    COMMAND          FUNCTION
    install          To install.
    start            To start.
    stop             To stop.
```
