####Installing:

    git clone https://github.com/HeitorG/nipe
    cd nipe
    cpan install  strict warnings Switch


####Commands:

	COMMAND          FUNCTION
	help             Displays the entire command list.
	install          For install.
	start            To start
	stop             To stop
	about            To know information about the developer
